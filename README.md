# atelier_4

"Ce projet est un exercice en Python qui consiste à gérer la relation entre
des employés et des voitures de service.
Chaque employé peut avoir une voiture et chaque voiture peut être
attribuée à un seul employé. Le programme permet d'afficher les informations des employés 
et des voitures, d'attribuer une voiture à un employé et de retirer une voiture.
